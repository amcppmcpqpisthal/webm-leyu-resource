# webm-leyu-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源形式存储，不针对任何特定域名或具体网站，主要供个人或团队作为资源备份、示例展示或快速部署使用。

仓库内容以 HTML 文件为主，结构清晰，适合直接通过 GitHub Pages 或其他静态托管服务进行访问。

## 目录说明

```
webm-leyu-resource/
├── index.html          # 首页或入口页面（可选）
├── pages/              # 存放多个独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放各个独立 HTML 页面，每个文件为一个完整页面，不依赖外部域名。
- `assets/`：存放页面共享的样式、脚本及图像资源，便于统一管理。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面间无强制依赖关系，可单独使用或组合展示。
- 资源文件使用相对路径引用，确保本地或静态托管环境下正常工作。
- 归档内容不包含任何外部链接、跟踪代码或营销信息。

## 维护说明

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任一 HTML 文件。
3. 或通过 GitHub Pages 等服务部署后访问对应路径。
